Explore listing grid

Explore listing grid

Develop dynamic listings and flexible grid layouts, so your CPTs, taxonomies, and User listings are displayed neatly and artfully. Insert and showcase all the requested meta-contents and see how cool it looks on the front end.

Grid

Customize the layout and number of columns to make the listing grid trend-perfect and eye-pleasing.

Masonry

Activate this mode to manage the columns’ height and give the listing a fresh modern look.

Injections

Got several listing items styled differently? Select Injections to merge the style settings and deliver your meta-contents as a unique layout.

Slider

Deliver the listing items as a set of scrollable slides. You can streamline navigation through arrows and dots or set up the autoplay mode.

Dynamic map listing

Benefit by the joint action of maps and listing grids. To learn more about a specific location, just click on the active image marker. You’ll be redirected to the existing CPT page or shown the location details in a popup window. Also, apply any filters you need.

Style settings

Style your map listing A to Z! Start by picking the desired map style, then customize the appearance of image markers. You can display each as an icon, image, or text.

Automatic adjustment

No more worries about manual data input. As soon as you add another custom post type, the marker with metadata will appear on the map automatically.

Give a try to map listing

Present listings as clickable points on the map. The setup is easy as ABC; just select items for the list and configure its queries. Map listing works best for travel agencies, booking services, and local diners.

Create custom calendar

Power your calendar with JetEngine’s dynamic widgets and work out a visually compelling schedule. Add as many events as you want and play around with the styles to make them catchy. As a result, you’ll get a classy calendar that ticks all of your boxes.

Get dynamic & data-packed

Tap into the dynamic widgets to build your calendar listing from scratch. Create multiple events per date and insert any content type you need. Icons, date and time, media, links, location, and even QR codes can be assigned to any given event.

Versatile styling options

Make your Calendar look fresh and stylish? It’s not a problem anymore. Select 1 of 4 available layouts, adjust the field size, background, typography, padding, and navigation arrows to your liking

Back-end display settings

Don’t forget to set the required query parameters such as the date, meta, and taxonomy query types to ensure a smooth display of the Calendar listing grid.
